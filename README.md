# Floating Ideas repository

The Floating Ideas website is built using [Jekyll](https://github.com/jekyll/jekyll) and hosted by [GitHub Pages](https://pages.github.com/).


This site uses the [Minimal Mistakes Jekyll theme](https://github.com/mmistakes/minimal-mistakes). You can [fork](https://github.com/mmistakes/mm-github-pages-starter/generate) the template repository or [fork](https://github.com/dgiaya/dgiaya.github.io/generate) this one to start your very own free blog.
